Introduction

L'informatique constitue la base fondamentale des systèmes informatiques modernes et des réseaux. Pour les professionnels de la cybersécurité, comprendre ces fondamentaux est essentiel pour :

  • Identifier efficacement les vulnérabilités dans les infrastructures IT
  • Mettre en œuvre des stratégies de protection adaptées et efficaces
  • Répondre rapidement et efficacement aux incidents de sécurité
  • Communiquer de manière technique avec les équipes réseau et développement
  • Concevoir des architectures sécurisées dès la phase de planification

Composants Matériels

  • CPU (Processeur) : Le "cerveau" de l'ordinateur qui exécute les instructions et traite les données. Les CPU interprètent et exécutent les instructions du programme, effectuent des calculs et contrôlent la plupart des fonctions du système.
  • Mémoire (RAM) : Stockage temporaire utilisé pendant que l'ordinateur fonctionne. La RAM fournit un accès rapide aux données dont le CPU a besoin activement, stockant les programmes en cours d'exécution et leurs données actives.
  • Périphériques de stockage : Stockage de données permanent incluant les disques durs (HDD), les disques SSD et le stockage réseau. Ces dispositifs conservent les données lorsqu'ils sont hors tension et fournissent un stockage à long terme pour le système d'exploitation, les applications et les fichiers utilisateur.
  • Périphériques : Appareils externes comme les claviers, souris, caméras et périphériques USB. Les périphériques étendent la fonctionnalité d'un ordinateur en fournissant des capacités d'entrée, de sortie ou de stockage supplémentaires.
  • Carte mère : La carte de circuit principal connectant tous les composants. La carte mère fournit les points de connexion physique et les voies de communication entre les composants, abrite le socket CPU, les slots mémoire et les connecteurs de cartes d'extension.
  • GPU (Unité de traitement graphique) : Processeurs spécialisés pour le rendu graphique et le calcul parallèle. Les GPU sont optimisés pour les opérations hautement parallèles nécessaires au rendu d'images, de vidéos et de graphiques 3D.

Équipements Réseau

  • Routeurs : Appareils qui dirigent le trafic entre les réseaux. Les routeurs connectent différents réseaux (comme votre réseau domestique à Internet) et déterminent le meilleur chemin pour que les données voyagent.
  • Commutateurs (Switches) : Appareils qui connectent plusieurs appareils au sein d'un même réseau. Les commutateurs créent des connexions dédiées entre les appareils sur un réseau local, améliorant l'efficacité en envoyant les données uniquement à leur destination prévue.
  • Pare-feu (Firewalls) : Appareils qui filtrent le trafic réseau. Les pare-feu surveillent et contrôlent le trafic réseau entrant et sortant en fonction de règles prédéterminées.
  • Points d'accès : Appareils qui fournissent une connectivité réseau sans fil. Les points d'accès permettent aux appareils de se connecter à un réseau câblé sans fil via des fréquences radio.
  • Modems et passerelles : Appareils qui connectent les réseaux aux fournisseurs de services Internet. Les modems convertissent le signal de votre FAI (câble, DSL, fibre) en un format utilisable par votre réseau.
  • Cartes d'interface réseau (NIC) : Composants matériels qui connectent les appareils aux réseaux. Les NIC fournissent l'interface physique pour les connexions réseau, soit via des ports Ethernet, soit via des antennes sans fil.

Logiciels & Installation

Catégories de Logiciels

  • Systèmes d'exploitation : Logiciel de base qui gère les ressources matérielles et fournit des services pour les applications (Windows, macOS, Linux). Les systèmes d'exploitation contrôlent toutes les opérations informatiques de base, fournissent une interface utilisateur et créent l'environnement dans lequel les applications s'exécutent.
  • Applications/Programmes : Logiciels conçus pour effectuer des fonctions ou des tâches spécifiques. Les applications fournissent une fonctionnalité spécialisée aux utilisateurs, des logiciels de productivité comme les traitements de texte aux outils créatifs, jeux et applications commerciales.
  • Utilitaires : Petits programmes qui effectuent des fonctions système spécifiques (nettoyeurs de disque, outils de sauvegarde). Les utilitaires aident à maintenir, optimiser ou étendre la fonctionnalité du système d'exploitation.
  • Pilotes (Drivers) : Logiciels qui permettent la communication entre le matériel et le système d'exploitation. Les pilotes fournissent la couche de traduction entre les périphériques matériels et le système d'exploitation, permettant au système d'exploitation de contrôler et d'interagir avec le matériel à l'aide d'interfaces standardisées.

Méthodes d'Installation

  • Supports physiques : Installation de logiciels à partir de DVD, clés USB ou autres supports physiques. Cela implique la configuration de la séquence de démarrage, la lecture des supports et le suivi des assistants d'installation.
  • Téléchargements numériques : Installation de logiciels obtenus depuis Internet. Cela inclut les téléchargements directs, les méthodes de vérification et le processus d'installation.
  • Gestionnaires de paquets : Utilisation d'outils centralisés d'installation et de gestion de logiciels :
    • Windows : Microsoft Store, Windows Package Manager (winget)
    • Linux : apt (Debian/Ubuntu), yum/dnf (RHEL/Fedora), pacman (Arch)
    • macOS : App Store, Homebrew

Licences Logicielles

  • Open source : Logiciel dont le code source est librement disponible et peut être modifié et redistribué selon des conditions spécifiques, favorisant la collaboration et la transparence.
  • Propriétaire : Logiciel commercial dont le code source reste fermé et dont l'utilisation est régie par des accords de licence stricts qui restreignent généralement la modification et la redistribution.
  • Types de licences : Différentes structures de licence existent pour répondre à divers besoins commerciaux. Les licences perpétuelles nécessitent un achat unique pour une utilisation indéfinie, tandis que les modèles d'abonnement impliquent des paiements récurrents pour un accès continu et des mises à jour.

Configuration Système

Configuration BIOS/UEFI

  • Objectif : Le BIOS/UEFI est le premier logiciel qui s'exécute lorsqu'un ordinateur est mis sous tension, initialisant le matériel avant le chargement du système d'exploitation.
  • Accès : Méthodes variables selon le fabricant (généralement touche F2, F10, F12 ou Suppr pendant le démarrage).
  • Paramètres courants : Ordre de démarrage, activation/désactivation du matériel, options de gestion de l'alimentation, paramètres de performance.

Partitionnement et Formatage

  • Partitionnement de disque : Division d'un périphérique de stockage physique en sections logiques séparées qui fonctionnent comme des lecteurs indépendants.
    • MBR (standard hérité, limité à 2 To par partition)
    • GPT (standard moderne, prend en charge des disques plus grands et plus de partitions)
  • Systèmes de fichiers : Structure utilisée pour organiser et stocker les données sur les partitions.
    • Windows : NTFS, FAT32, exFAT
    • Linux : ext4, XFS, Btrfs
    • Multiplateforme : exFAT

Configuration Réseau

  • Adressage IP :
    • IP statique : Adresse réseau fixe attribuée manuellement
    • DHCP : Adresse attribuée automatiquement par un serveur central
  • Configuration DNS : Mise en place de serveurs DNS qui traduisent les noms de domaine lisibles par l'homme en adresses IP.
  • Outils de connectivité : Programmes comme ping, traceroute/tracert, ipconfig/ifconfig et nslookup qui aident à diagnostiquer les problèmes réseau.

Sécurité de Base

  • Pare-feux : Systèmes de filtrage du trafic réseau qui contrôlent les connexions autorisées en fonction de règles.
  • Antivirus/Antimalware : Logiciels qui détectent et suppriment les programmes malveillants qui pourraient nuire au système ou compromettre les données.
  • Paramètres de sécurité standards : Mises à jour automatiques, politiques de compte (exigences de mot de passe), délais d'expiration du verrouillage d'écran.

Solutions de Sauvegarde

  • Types de sauvegarde :
    • Complète : Copie complète de toutes les données sélectionnées
    • Incrémentielle : Modifications depuis la dernière sauvegarde
    • Différentielle : Modifications depuis la dernière sauvegarde complète
  • Options de stockage : Disques externes locaux, stockage réseau ou services cloud.
  • Planification : Définition d'intervalles réguliers pour les sauvegardes automatisées.

Système Binaire (Base-2)

  • Base fondamentale : Utilise uniquement deux chiffres (0 et 1) pour représenter toutes les données et instructions.
  • Terminologie : Bit (chiffre binaire unique), Octet (8 bits), Nibble (4 bits).
  • Applications : Adresses IP, masques de sous-réseau, adresses MAC, permissions de fichiers, encodage de caractères.

Système Octal (Base-8)

  • Base-8 : Utilise les chiffres 0-7, chaque chiffre octal = 3 bits binaires.
  • Permissions Unix/Linux : Permissions de lecture(4), écriture(2), exécution(1).
    • Valeurs courantes : 755 (rwxr-xr-x), 644 (rw-r--r--)
    • Commande chmod : chmod 755 nomfichier

Système Décimal (Base-10)

  • Système standard : Utilise dix chiffres (0-9), le plus intuitif pour le comptage humain.
  • Applications IT : Interfaces utilisateur, valeurs de configuration (numéros de port), métriques de performance, adressage réseau en notation décimale pointée.

Système Hexadécimal (Base-16)

  • Base-16 : Utilise les chiffres 0-9 et les lettres A-F (A=10, B=11...F=15).
  • Relation binaire : Chaque chiffre hexadécimal = 4 bits binaires.
  • Applications IT : Adresses mémoire, codes couleur (#FF0000), adresses MAC, adresses IPv6, valeurs de hachage.

Encodage de Caractères

  • ASCII : Standard 7 bits (128 caractères), caractères de base (lettres, chiffres, ponctuation).
  • Unicode (UTF-8/16) : Standard universel prenant en charge toutes les langues du monde, compatible avec ASCII, standard web pour HTML, JSON, XML.
  • Base64 : Schéma d'encodage binaire-en-texte utilisant 64 caractères ASCII imprimables. Applications : pièces jointes email, API web, certificats SSL/TLS, jetons JWT.
