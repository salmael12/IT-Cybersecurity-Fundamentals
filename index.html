1️⃣ Introduction

L'informatique est la base fondamentale de la cybersécurité moderne. Une compréhension approfondie du matériel, des réseaux et des systèmes d'exploitation permet aux professionnels de :

  • Détecter et analyser les vulnérabilités dans les infrastructures IT
  • Implémenter des stratégies de protection efficaces et adaptées
  • Répondre rapidement et efficacement aux incidents de sécurité
  • Communiquer de manière technique avec les équipes réseau et développement
  • Concevoir des architectures sécurisées dès la phase de planification

Les équipements réseau sont les composants physiques essentiels qui permettent la communication entre les dispositifs dans une infrastructure informatique. Voici les principaux équipements utilisés dans les réseaux modernes avec leurs caractéristiques techniques détaillées :

Routeurs (Routers)

  • Fonction principale : Connecter différents réseaux (LAN/WAN) et diriger le trafic entre eux en utilisant des tables de routage dynamiques ou statiques
  • Couche OSI : Couche 3 (Réseau)
  • Protocoles : RIP, OSPF, BGP, EIGRP
  • Utilisation typique : Réseaux d'entreprise, fournisseurs d'accès internet (FAI), passerelles internet
  • Exemples : Cisco ISR 4000, Juniper MX Series, MikroTik RouterOS

Commutateurs (Switches)

  • Fonction principale : Connecter les appareils au sein d'un même réseau local (LAN) et transmettre les trames de données uniquement à la destination correcte en utilisant les adresses MAC
  • Couche OSI : Couche 2 (Liaison de données) - Les switches multicouches fonctionnent aussi en couche 3
  • Technologies : VLAN, STP, PoE (Power over Ethernet), LACP
  • Utilisation typique : Bureaux, centres de données, réseaux locaux d'entreprise
  • Exemples : Cisco Catalyst 9000, HPE Aruba, Dell PowerSwitch

Pare-feu (Firewalls)

  • Fonction principale : Filtrer le trafic entrant et sortant selon des règles de sécurité prédéfinies pour protéger le réseau contre les menaces externes et internes
  • Couche OSI : Couches 3-7 (Réseau à Application)
  • Types : Pare-feu stateful, pare-feu nouvelle génération (NGFW), pare-feu applicatifs (WAF)
  • Fonctionnalités : Inspection DPI (Deep Packet Inspection), IPS/IDS, filtrage d'URL, contrôle d'applications
  • Utilisation typique : Périmètre réseau, segmentation interne, protection des serveurs
  • Exemples : Fortinet FortiGate, Palo Alto Networks, Cisco Firepower

Points d'accès sans fil (Wireless Access Points)

  • Fonction principale : Permettre aux appareils sans fil de se connecter au réseau câblé via des ondes radio (Wi-Fi)
  • Couche OSI : Couches 1-2 (Physique/Liaison)
  • Standards : IEEE 802.11a/b/g/n/ac/ax (Wi-Fi 6)
  • Technologies : MIMO, beamforming, roaming rapide, gestion centralisée
  • Utilisation typique : Environnements mobiles, bureaux modernes, campus universitaires, hôtels
  • Exemples : Cisco Aironet, Aruba AP, Ubiquiti UniFi

Modems et Passerelles

  • Modem : Convertit le signal analogique (DSL/câble) ou optique (fibre) en signal numérique utilisable par les équipements réseau
  • Passerelle : Combine les fonctions de modem, routeur, switch et parfois point d'accès en un seul appareil
  • Couche OSI : Couche 1 (Physique)
  • Utilisation typique : Connexion internet chez les particuliers et PME
  • Exemples : Box opérateur (Orange Livebox, SFR Box), modems fibre Huawei

Serveurs

  • Fonction principale : Fournir des services centralisés comme le stockage, l'hébergement web, l'authentification, ou la virtualisation aux clients du réseau
  • Couche OSI : Couche 7 (Application)
  • Types : Serveurs web, serveurs de fichiers, serveurs de base de données, serveurs d'application
  • Caractéristiques : Haute disponibilité, redondance, gestion à distance (IPMI/iLO)
  • Utilisation typique : Centres de données, infrastructures critiques, cloud computing
  • Exemples : Dell PowerEdge, HPE ProLiant, Lenovo ThinkSystem

Autres équipements essentiels

  • Répéteurs (Repeaters) : Amplifient le signal réseau pour étendre la portée physique (couche 1)
  • Ponts (Bridges) : Connectent deux segments de réseau au niveau de la couche 2
  • Concentrateurs (Hubs) : Anciens équipements qui diffusent le trafic à tous les ports (obsolètes, remplacés par les switches)
  • Cartes réseau (NIC) : Interfaces physiques intégrées ou externes permettant la connexion réseau (Ethernet, Wi-Fi, fibre)
  • Équipements de téléphonie IP : Téléphones IP, contrôleurs PBX pour la voix sur IP

4️⃣ Logiciels & Installation

  • Systèmes d'exploitation : Windows Server 2022, Ubuntu Server 22.04 LTS, pfSense
  • Applications réseau : Wireshark, Nmap, Metasploit, Burp Suite

5️⃣ Sécurité système

  • Gestion des identités : Active Directory, LDAP, RBAC, moindre privilège
  • Pare-feu applicatif : Windows Defender Firewall, iptables, pfSense
  • Antivirus/EDR : CrowdStrike, SentinelOne, Microsoft Defender for Endpoint
  • Chiffrement : BitLocker, LUKS, TLS 1.3, certificats X.509
  • Mises à jour : WSUS, Landscape, patch management automatisé
  • Sauvegardes : Veeam, Bacula, stratégie 3-2-1-1-0 pour la résilience
