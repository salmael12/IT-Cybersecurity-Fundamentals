Introduction

Comprendre les bases de l'informatique et les systèmes de numération est essentiel pour débuter et manipuler les données correctement.

Matériel

  • CPU : Processeur
  • RAM : Mémoire vive
  • Disques : Stockage HDD/SSD
  • Périphériques : Clavier, souris, écran

Logiciels

  • Systèmes d'exploitation : Windows, Linux, macOS
  • Applications : Word, navigateur, jeux
  • Pilotes : Permettent au matériel de communiquer avec le système

Réseau

  • Routeur : Connecte plusieurs réseaux
  • Switch : Connecte plusieurs appareils sur un réseau local
  • Points d'accès : Connexion Wi-Fi

Binaire (Base 2)

Chiffres : 0 et 1. Chaque chiffre = 1 bit. 8 bits = 1 octet.

Exemple : 1011₂ = 11₁₀

Octal (Base 8)

Chiffres : 0 à 7. Chaque chiffre = 3 bits. Utilisé pour permissions Linux.

Décimal (Base 10)

Chiffres : 0 à 9. Utilisé dans la vie quotidienne.

Hexadécimal (Base 16)

Chiffres : 0-9 et lettres A-F. Chaque chiffre = 4 bits.

ASCII

Chaque caractère a un code numérique (0 à 127).

Exercices pratiques

1️⃣ Convertir le binaire 1101 en décimal :

Solution : 1101₂ = 8 + 4 + 0 + 1 = 13₁₀

2️⃣ Convertir le décimal 14 en binaire :

Solution : 14 ÷ 2 = 7 reste 0, 7 ÷ 2 = 3 reste 1, 3 ÷ 2 = 1 reste 1, 1 ÷ 2 = 0 reste 1 → 1110₂

3️⃣ Valeur ASCII de la lettre B :

Solution : B = 66 décimal = 1000010₂
